What can Pig Dissection VR Do?

Pig Dissection VR currently offers 2 distinct activities for users to enjoy

Teachers can give their student a unique class code to use at the beggining of the activity so progress can be tracked

Activity 1

Activity 1 requires the user to make external measurements of the pig, to determine age and gender.

The user will demonstrate proper hygine and saftey precautions, and will learn about external qualities to perform basic pig examination

Features

  • Hygine and Saftey precautions

  • Measuring External Features

  • Use of VR tools

  • Basic Classification of age and gender

  • Step by step guide on the web

Activity 2

Activity 2 requires the user to make internal incisions to identify and examine anatomical parts of the pig.
Along with the proper saftey precautions, the user will have to make incisions on the pig, and identify key organs and components in different body systems.

Features

  • Hygine and Saftey precautions

  • Measuring External Features

  • Use of VR tools

  • Incising into the mouth and throat system

  • Basic Classification of age and gender

  • Step by step guide on the web
